I design and ship **backend systems** (Laravel, Symfony, REST), **data layers** (MySQL, PostgreSQL, Redis), and **mobile clients** (Kotlin/Android, Swift/iOS, Flutter) — with Docker, GitLab CI/CD, and clear API contracts.
