### About me

```text
Senior engineer · iOS & mobile-first · strong PHP backend depth · Germany
```

I design and ship **REST APIs** (Laravel, Symfony), **data layers** (MySQL, PostgreSQL, Redis), and **native mobile clients** (Swift/iOS, Kotlin/Android) — with Docker, GitHub Actions, OpenAPI contracts, and test-driven delivery.

I've led mobile teams, published to the **App Store** and **Google Play**, and I build portfolio samples that mirror how production teams work: CI, layered architecture, and clear READMEs.
