### About

I design and ship **native iOS apps** (Swift, SwiftUI, UIKit) and **PHP/Laravel/Symfony REST APIs** — end to end, from API contracts and database performance to App Store delivery.

I've shipped to the **App Store** and **Google Play**, led mobile teams, and build portfolio repos the way production teams do: layered architecture, tests, Docker, CI, and clear documentation.
